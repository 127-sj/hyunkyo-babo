# 제품 요구사항 정의서 (PRD) - 임현교 키우기 (다마고치 게임)

## 1. 개요 (Overview)
'임현교 키우기'는 웹 브라우저에서 가볍게 즐길 수 있는 방치형/육성 다마고치 게임입니다. 사용자는 귀여운 캐릭터(임현교 돼지)에게 먹이를 주고, 씻겨주고, 재우는 일상 활동과 더불어, 알바 미니게임을 통해 돈을 벌어 상태를 관리해야 합니다. 캐릭터의 상세 수치에 따라 외형과 표정이 동적으로 변하는 시각적 피드백이 이 게임의 핵심 요소입니다. 21살 성인 달성 시 클리어됩니다.

## 2. 목표 (Goals)
* 사용자에게 시각적이고 즉각적인 피드백을 주며 가상의 캐릭터를 애정으로 키우는 재미를 제공.
* 배고픔, 청결도, 에너지, 행복도 등 4가지 스테이터스의 균형을 맞추도록 유도.
* 알바 미니게임과 상점 경제 시스템을 통해 간단한 재화 획득 및 소비 경험(루프) 제공.

## 3. 핵심 기능 (Key Features)

### 3.1. 캐릭터 시각 효과 동적 변화 시스템
* **성장 (Age):** 0살에서 시작하여 시간(Tick) 경과에 따라 나이가 증가합니다. 특정 연령대(5살, 13살, 20살) 돌파 시 캐릭터의 성장 라벨과 스케일(Scale)이 더 커집니다.
* **배고픔 (Hunger):** 수치가 낮아질수록 CSS 변환 효과를 통해 캐릭터의 몸통 가로 너비가 홀쭉해집니다. 일정 수준 이하로 떨어지면 아사(게임 오버)합니다.
* **청결도 (Cleanliness):** 수치가 낮아질수록 몸 주변이 더러워지는 세피아 톤 필터가 씌워집니다. 병에 걸려 사망(게임 오버)할 수 있습니다.
* **에너지 (Energy):** 에너지가 떨어질수록 눈꺼풀이 처지며 피곤한 눈 모양으로 변합니다.
* **행복도 (Happiness):** 낮아질 경우 웃는 입꼬리가 역전되어 시무룩하게 쳐진 표정으로 바뀝니다.
* **대사 (Speech Bubble):** 현재 가장 부족한 스탯에 맞춰 캐릭터 머리 위의 말풍선 내용이 무작위 패턴으로 갱신됩니다. 사용자 클릭(Poke) 시에도 반응합니다.

### 3.2. 상호작용 및 액션 (Actions)
* **음식 상점 (밥 구매/먹이기):** 상점 모달 팝업에서 보유한 돈(원)을 지불하여 3가지 음식 옵션(사과, 햄버거, 프리미엄 고기) 중 하나를 선택해 먹여 배고픔과 행복도를 회복시킵니다.
* **씻기기:** 캐릭터의 청결도를 단번에 회복시킵니다.
* **재우기:** 수면 모드로 전환합니다. 캐릭터가 쿨쿨 자는 표정으로 바뀌고 배경이 어둡게 변하며 주기적으로 Zzz 이펙트가 발생합니다. 수면 중 에너지가 점진적으로 회복되며 100% 도달 시 자동 기상합니다.
* **놀기:** 행복도를 대폭 상승시키지만 체력(에너지)을 크게 소모하고, 땀을 흘리므로 청결도도 일부 감소합니다.
* **알바가기:** 화면에 광클 미니게임 팝업창이 활성화됩니다. 클릭을 통해 업무 게이지를 올리며 에너지를 소모해 완료 시 10~20원의 돈을 무작위로 획득합니다. 

### 3.3. 경제 시스템
* 초기 시작 시 50원의 기본 자본이 주어집니다.
* '알바가기' 노동을 통해 지속적으로 재화를 획득합니다.
* '음식 상점'에서 재화를 소비합니다 (사과 10원, 햄버거 30원, 스테이크 50원).

## 4. 기술 스택 (Tech Stack)
* **HTML 구조:** 시맨틱 마크업을 지향하며 게임 인터페이스 요소를 모달, 그리드, 컨테이너로 조립.
* **CSS 패키징 (Tailwind CSS + Custom CSS):**
    * Tailwind CSS(CDN)를 활용하여 모달 팝업, 헤더 상태바, 버튼 레이아웃, 컬러를 신속하게 구성.
    * Vanilla CSS를 활용해 캐릭터 파츠 구조화(CSS Figure), 애니메이션(floating, bounce flyUp 이펙트), 런타임 CSS 속성(filter, transform-origin)을 제어하여 역동적인 시각 변화 구현.
* **JavaScript 로직:** `requestAnimationFrame` 사이클을 이용해 게임 루프를 메인 스레드에 부담없이 구현, 상태(State) 데이터 바인딩 및 DOM/이벤트 제어.

## 5. UI/UX 디자인 (Design Aesthetics)
* 전반적인 테마 환경은 파스텔 핑크(#fce4ec) 기반의 따뜻하고 포근한 느낌 제공.
* 둥근 컨테이너 테두리 표면 반사와 그림자 효과(Drop Shadow)로 편안한 입체감.
* 폰트는 'Jua' 구글 웹 폰트를 채용하여 게임 화면의 동글동글한 캐주얼 느낌을 강화.
* 모든 인터랙션 수행 시 팝업 아이콘 이펙트(사과, 비누방울, 동전 튀어오름 등)가 활성화되어 확실한 인터랙션 타격감을 사용자에게 전달.
